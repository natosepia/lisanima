# rulebookマイグレーション計画

## 概要

CLAUDE.md（グローバル + プロジェクト）および既存 t_rulebooks のルールを、
新しい m_rulebooks（Materialized Path構造）に統合・移行する。

- CLAUDE.md → ブートローダー化（接続指示 + 最低限フォールバックのみ残す）
- m_rulebooks → ルールのSSoT

## 凡例

- **persona_id**: null=該当なし（見出し行） / `*`=全ペルソナ共通 / `リサ`=リサ専用
- **is_editable**: F=FALSE（なとせ管理） / T=TRUE（リサ書き換え可）
- persona_id の詳細定義は docs/04_schema.md に記載（SSOT原則1）
